# 🚀 Coding Fundamentals & DSA Journey

Welcome to my central repository for learning programming! This workspace tracks my daily progress from basic syntax to advanced problem-solving, serving as a personal portfolio of my coding journey.

## 🎯 Objective
The goal of this repository is to build a strong foundation in computer science concepts by writing code in multiple languages, gradually scaling up to Data Structures and Algorithms (DSA), and building small projects.

## 📂 Repository Structure

This repository is organized progressively to reflect my learning path:

### `01_Hello_World/`
The starting point. Basic input/output programs to understand the compilation and execution process across different languages.
* Languages covered: C, C++, Java, Python

### `02_Fundamentals/`
Core programming constructs and logic building.
* Control Flow (If/Else, Switch cases)
* Loops (For, While, Do-While)
* Functions and Recursion
* Arrays and Strings (1D/2D manipulations)

### `03_Data_Structures/`
Implementation of core data structures from scratch.
* Linked Lists (Singly, Doubly, Circular)
* Stacks and Queues
* Trees and Graphs
* Hash Tables

### `04_Algorithms/`
Problem-solving and algorithmic thinking.
* Searching (Linear, Binary)
* Sorting (Bubble, Merge, Quick, Insertion)
* Two-Pointer Technique & Sliding Window

### `05_Mini_Projects/`
Small, practical applications applying the concepts learned in the previous stages.

## 🛠️ Languages & Tools Used
* **C / C++:** For understanding memory management and competitive programming.
* **Java:** For Object-Oriented Programming (OOP) principles.
* **Python:** For scripting, rapid prototyping, and basic algorithms.
